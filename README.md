## 大致思路：
> 注：大部分的代码解释已经在各个版本的代码源文件中做了标注

1.代码整体采用了面向过程的编程范式，用函数完成基本的运行过程<br>
2.main()函数的主体采用了一个死循环，使得程序始终运行，并且设置了 触墙死亡 的退出机制。<br>
3.在死循环运行前，通过一个数组来定义并绘制整个地图，并确定坐标的判定规则<br>
4.使用一个数组来定义蛇体（压缩内存可以用链表，这里没有采用），通过死循环的间隔执行来不断刷新蛇体的状态，从而实现蛇体前进<br>
5.通过Windows.h下的键盘句柄来读取控制台的键盘输入，并通过干涉数组的调用来实现蛇体的转向<br>
6.使用随机值实现食物的生成，并通过枚举来标记，使得程序更具备可读性<br>
